----- Switch Weapons -----

R - Rifle
V - Pistol 
F - Flashlight
C - Knife

----- Movement System -----
W - Forward
A - Left
S - Back
D - Right

----- Attack System -----
Left Click - Melee Attack
Shift + Left Click - Shoot

----- Aiming System -----
Hold Right Click + Move Mouse - Adjust Aim Direction
